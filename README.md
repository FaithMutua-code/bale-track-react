## ğŸ“Œ Project Overview

**BaleTrack** is a modern stock management web application designed for
mitumba traders, boutique owners, and small retail businesses.\
It allows you to track bales, warehouse stock, expenses, sales,
purchases, and savings --- all in one clean dashboard.

> Note: The system is fully adapted for mitumba bales, not agricultural
> cotton.

---

Feature Description

---

Dashboard Analytics Real-time overview of stock, profit,
expenses & sales

Bale Data Entry Add bale type (Jeans, T-Shirts, Camera,

## âš™ï¸ Setup & Installation

### Prerequisites

- Node.js v18+
- MongoDB (Local or Atlas)
- Git

### Clone the Project

    git clone https://github.com/yourusername/bale-track.git
    cd bale-track

### Install Dependencies

    npm install

### Set Up Environment Variables

Copy .env.example to .env:

    cp .env.example .env

Edit:

    VITE_MONGODB_URI=mongodb://localhost:27017/baletrack

### Start Development Server

    npm run dev

Open in browser: http://localhost:5173
